Em & Rem are similars but they are multipliers of different things:

  • Em unit is a multiplier relative to the font-size of it's container or parent unit
  • Once we add a font-size to the actual element itself, the em unit will go on that. ex header p { font-size: 1.5em; //computed value: 30px
    padding: 1em; //computed value: 30px instead of 20px which mentioned in the header (header { font-size: 20px; })}
  • Another thing is when dealing with nested elements like ul and lis
  • Rem unit is relative to the font-size of the root html element. It default font-size is 16px which can be modified within CSS
